Python Flask Application

Note: You requested a Python Flask application. Flask is a backend framework that runs on a server, not directly in a web browser. The code below must be saved in a Python file (`.py`) and run from your terminal.

File: app.py

from flask import Flask

app = Flask(__name__)

@app.route('/status')
def status():
    return 'OK'

if __name__ == '__main__':
    app.run(debug=True)

How to Run This App

  1. Save the code block above into a file named app.py.
  2. Open your terminal or command prompt and install the Flask library if you don't have it:
    pip install Flask
  3. In your terminal, navigate to the directory where you saved app.py and run the application:
    python app.py
  4. Your server is now running. Open your web browser and go to the following URL to see the 'OK' response:
    http://127.0.0.1:5000/status
