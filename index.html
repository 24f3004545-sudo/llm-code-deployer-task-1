Simple Python Flask App

This page provides the code for a simple Python Flask application. The app creates a single endpoint /status that returns 'OK'.

Setup Instructions

  1. Ensure you have Python and pip installed on your system.
  2. Install Flask via the terminal: pip install Flask
  3. Save the code below as a Python file (e.g., app.py).
  4. Run the app from your terminal: python app.py
  5. In your browser, navigate to http://127.0.0.1:5000/status to see the result.
app.py 
from flask import Flask

app = Flask(__name__)

@app.route('/status')
def status():
    return 'OK'

if __name__ == '__main__':
    # Runs the app on a local development server.
    app.run(debug=True)
